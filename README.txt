このフォルダのデータは、Shibuya.tracの皆様へ提供させていただいたロゴの元データを含みます。
ただし、最初に投稿させていただいたデータは、誤って削除してしまいました。
このため、アップしていたPNGをもとに、できるだけ近くなるように再現、再作成したものです。

白の斜線や足跡のラインはどうしてもオリジナルの再現ができませんでした。
近いものになるように努めましたが、厳密な意味で、皆さんに選んでいただいたものとは異なります。
公正さを要するのであれば、採用を取り下げて下さってもかまいません。

なお、データの加工や2次利用は特に断り無く行って下さってかまいません。
Illustratorで作成していますが、*.ai はバージョン10形式で保存していますので、少し前の世代の
ものでも開けると思います。
レイヤー情報を保持していますので、文字や足跡のレイアウト、サイズ調整も必要に応じて行って下さい。
（BDSとかライセンスには疎いので、細かいことは書きません...）
